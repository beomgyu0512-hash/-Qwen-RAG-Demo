# 光刻工艺概述

光刻是晶圆制造中将电路图形从掩膜版转移到晶圆表面的关键步骤，通常与涂胶、曝光、显影和刻蚀等环节协同完成。它的目标是在晶圆上形成高精度图案，为后续的离子注入、刻蚀和薄膜沉积提供结构基础。

典型流程包括：

1. 在晶圆表面均匀旋涂光刻胶。
2. 通过预烘提升胶膜稳定性。
3. 使用曝光设备将掩膜版图形投射到光刻胶上。
4. 经过显影后保留目标图形。
5. 将图形转移到下层材料，最后去胶清洗。

光刻工艺的核心指标通常包括分辨率、套刻精度、线宽均匀性和缺陷率。随着制程节点缩小，曝光波长、数值孔径和工艺窗口控制都会直接影响成品率。

在实际制造中，光刻不仅影响单层图形质量，还会影响多层叠加时的对准误差。如果套刻偏差过大，可能导致后续金属互连短路、开路或器件性能漂移。
