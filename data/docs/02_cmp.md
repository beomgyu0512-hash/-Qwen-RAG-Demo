# CMP 化学机械抛光

CMP 是 Chemical Mechanical Polishing 的缩写，常译为化学机械抛光。它通过化学反应与机械研磨的协同作用，对晶圆表面进行全局平坦化处理。

CMP 的主要作用包括：

1. 消除前序沉积或刻蚀后形成的表面高低起伏。
2. 为后续光刻提供更平整的基底，降低景深和聚焦误差带来的风险。
3. 在铜互连工艺中去除多余金属与阻挡层，只保留沟槽和通孔中的目标材料。

CMP 过程通常涉及抛光垫、抛光液、载具压力和转速等参数。若抛光过度，可能出现 erosion 或 dishing 等问题；若抛光不足，则会影响后续图形转移和层间连接可靠性。

因此，CMP 常与端点检测、片内均匀性监控以及后清洗流程联合优化，以兼顾平坦度、去除速率和表面缺陷控制。
