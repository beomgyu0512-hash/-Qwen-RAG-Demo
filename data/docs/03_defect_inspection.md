# 半导体缺陷检测方法

缺陷检测是晶圆制造质量控制的重要组成部分，目标是在尽可能早的工序阶段发现颗粒污染、图形异常、划伤、残留、桥连或断线等问题。

常见方法包括：

1. 光学检测：利用明场、暗场或散射信号识别表面和图形缺陷，适合大面积快速扫描。
2. 电子束检测：使用更高分辨率对微小缺陷进行复检和分类，但吞吐量通常较低。
3. AOI 自动光学检测：结合图像处理算法，对规则图形和重复单元进行比对。
4. 缺陷复判与分类：通过规则引擎或机器学习模型对缺陷进行归因，区分工艺波动与真实失效风险。

在先进制程中，缺陷检测不仅关注是否有异常，还要关注缺陷的尺寸、位置、重复性与对电性能的潜在影响。只有将检测结果与工艺段、设备、批次和 SPC 数据联动，才能真正支持良率改善。
