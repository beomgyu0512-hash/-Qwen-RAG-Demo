# 开发记录

## 项目定位

该项目是一个面向智能制造场景的轻量级 RAG Demo，重点验证本地知识库问答流程，而不是实现完整的平台化系统。默认资料围绕半导体制造、缺陷检测、SPC 和先进封装展开。

## 关键实现

- 完成文档加载、文本切分、Embedding、向量入库、Top-K 检索与回答生成
- 使用 Chroma 作为本地向量库
- 使用 Gradio 提供演示界面
- 补充了 24 条测试问答和批量评测脚本

## 主要工程问题

### Python 3.9 兼容

- 移除了 `dataclass(slots=True)` 和 `zip(..., strict=True)` 这类高版本写法
- 使用工作区本地缓存执行 `compileall`，避免系统缓存目录权限问题

### 依赖兼容

- 将 `gradio` 约束在 `4.44.x`
- 显式加入 `huggingface_hub<1.0`，解决 `HfFolder` 导入问题

### Gradio 页面稳定性

- 将上传组件调整为 `gr.File(type="filepath")`
- 为事件回调设置 `show_api=False`，避开部分 schema 生成兼容问题

### 启动与本地部署

- 启动逻辑按顺序尝试多个候选端口，避免固定端口被占用
- 页面默认围绕智能制造文档问答场景组织，但上传接口本身支持任意 `md/txt/pdf/docx` 文件

## 当前状态

- 默认示例资料可直接建库
- 批量评测脚本可生成 `csv + md` 结果
- 仓库内容已整理为适合公开展示的结构
